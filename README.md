# 🚀 TechBlog - Plataforma de Gestión de Contenidos

TechBlog es una plataforma de administración y publicación de artículos con autenticación JWT, gestión de usuarios y control de contenido. Permite a los usuarios autenticados crear, editar y eliminar sus propios artículos en un entorno seguro y dinámico.

## 🔥 Características
✅ **Autenticación y Autorización con JWT** - Protege rutas y gestiona sesiones de usuario.  
✅ **Gestión de Usuarios** - Registro, inicio de sesión y control de sesiones.  
✅ **Publicación y Edición de Artículos** - Creación, edición y eliminación de posts con asignación automática del autor.  
✅ **Panel de Administración** - Interfaz interactiva con Bootstrap para una mejor experiencia.  
✅ **Carga Dinámica de Contenido** - Sin necesidad de recargar la página.  
✅ **Gestión de Imágenes** - Almacenamiento de imágenes de perfil y artículos.  
✅ **Restricción de Acceso** - Los usuarios solo pueden modificar sus propios posts.  
✅ **Manejo de Errores y Notificaciones** - A través de modales interactivos y alertas.  

## 🛠️ Tecnologías Utilizadas
- **C# & .NET Core** - Backend y API RESTful.  
- **MongoDB** - Base de datos NoSQL para almacenar usuarios y artículos.  
- **JWT (JSON Web Token)** - Seguridad y autenticación.  
- **HTML, CSS & JavaScript** - Desarrollo frontend.  
- **Bootstrap** - Interfaz responsive y estilización.  
- **Fetch API** - Consumo de la API REST.  

## 🎨 Diseño Responsive  
El diseño de la plataforma está optimizado para funcionar en cualquier dispositivo, con un enfoque en la usabilidad y la experiencia del usuario.  

## 📌 Funcionalidades Clave
- **Login y Registro con JWT**  
- **Panel de Administración Dinámico**  
- **Publicación de Artículos con Validación**  
- **Gestión de Imágenes y Archivos Adjuntos**  
- **Carga y Eliminación de Contenidos**  

## 🚀 Despliegue  
Este proyecto está diseñado para ser ejecutado en un entorno local y preparado para despliegues en servidores con MongoDB y .NET Core.
